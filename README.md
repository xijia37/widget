======

###load.fn.js 介绍###

>此插件主要作用是用来模仿‘可编辑下拉框’，与联动功能相似。
>
>在需要应用到的input引用linkAge，并添加参数，即可生成所需的下拉列表
>
>此插件的参数如下所示
+ dataDiv ：数据ul的外部div，可由divHeight控制高度
+ dataCurrent ：当前选中数据li的样式
+ dataPointer ：数据指向
+ dataSynchronous ：同步数据的id到另一dom元素中
+ ajaxUrl ：'ajax地址 key='
+ divHeight:'div的高度 li的倍数，默认8'
